第四节 幼儿园常见问题处理
考点
1：外伤

小外伤

切割伤

用干净纱布按压伤口止血，止血后，在伤口周围用75%的酒精由里向外消毒，敷上消毒纱布，用绷带包扎。

 刺伤

1. 用生理盐水或冷开水清洗伤口。

2. 如有异物，用消毒针将其拔除。

3. 用碘酒或酒精涂抹伤口周围，伤口处涂红药水。

扭伤

检查是否骨折；如无骨折，立即冷敷，收缩止痛；一天后，热敷，减轻肿胀。

骨折

急救

在未包扎前，不能移动伤者；如有大出血，在伤口处覆盖敷料，止血，使断骨不再刺痛周围组织，限制受伤肢体活动。

预防

1. 保教人员加强责任心，防止发生伤害事故。

2. 幼儿进出门不安装弹簧，以免夹伤幼儿。

3. 教育幼儿不做危险动作。

出血

急救

动脉出血：采用指压止血法，用单个或多个手指压住血管的上端，急送医处理。

小伤口：用一般止血法，用纱布、棉花垫在伤口，绷带包扎。

较大伤口：将敷料盖在伤口，绷带包扎。

预防

1. 将小刀等利器放在幼儿拿不到的地方。

2. 经常检查幼儿口袋，将危险物品交由老师保管。

3. 教育幼儿不用带尖带刺的东西做玩具、挖鼻孔。

4. 注意幼儿活动安全。

烧烫伤

急救

•  消除烧、烫伤原因

•  保护创面

•  设法使伤员安静止痛

预防

1. 成人端热水注意避开幼儿。

2. 开水、烫饭、化学物品等远离幼儿。

3. 饭菜凉后再让幼儿进食。

4. 教育幼儿不玩火、不触摸电器等物品。
