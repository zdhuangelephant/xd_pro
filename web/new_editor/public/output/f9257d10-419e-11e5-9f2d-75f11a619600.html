第二节 教师职业行为
考点
2：中学教师的职业特点

专业特征

•  具有一定的职业声望

•  需要严格职前专业训练

•  具有专业自主性

•  具有自己的专业标准

•  注重实践 

职业角色的特征

教师职业最大的特征就是多样化，职业角色主要有：

1. 传道者角色

教师的道德观、价值观总是代表着社会主导地位的道德观、价值观，并用这种观念引导学生。

2. 授业解惑者角色

教师掌握了知识经验和技能，通过对其加工整理后，帮助青少年解决他们的困惑。 

3. 示范者的角色

教师的言行是学生学习和模仿的榜样，教师的为人处世的态度对学生有潜移默化的影响。

4. 管理者的角色

教师是学校教育教学活动的管理者，要肩负起教学管理的职责。

5. 朋友的角色

学生希望得到老师在学习、生活、人生等多方面的额知道，希望于老师分享快乐与忧伤。

6. 研究者的角色

教师要以一种变化发展的态度来对待自己的工作对象、工作内容，不断学习、不断反思。

 教师职业的劳动特征

1. 复杂性

指教师的劳动不是一种简单的劳动，而是一种复杂劳动。具体体现在，教育目的的全面性；教育任务的多样性；劳动对象的差异性。

2. 创造性

主要表现在教师创造性的运用教育教学规律，比如，因材施教，教学方法上的不断更新以及教师需要“教育机智”等。

3. 主体性和示范性

主体性是指教师自身可以成为活生生的教育因素和具有影响力的榜样。示范性是指教师的言行举止，包括人品、才能、治学态度等都会成为学生学习的对象。

4. 劳动时间和空间的延展性

劳动时间的延续性是由于教师没有固定的工作时间长度，也没有严格的交接班时间界限。空间的广延性是指教师没有严格界定的劳动场所，这是由影响学生发展因素的多样性决定的。

5. 长期性和间接性

长期性是指人才培养的周期比较长，教育影响具有滞后性。间接性是指教师的劳动不直接创造物质财富，而是以学生为中介实现教师劳动的价值。

6. 个体性与群体性

劳动手段角度，教师的劳动主要是以个体劳动的形式进行。同时教师的劳动成果又是集体劳动和多方影响的结果。
