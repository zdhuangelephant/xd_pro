第一节 阅读理解能力
考点
2：对文章的归纳分析

分析结构

记叙文

1. 记事文：从事情发展的先后时间、地点转换、阶段入手分析

2. 记人文：从人物成长的阶段、人物所在地点转换、人物情感变化入手分析

3. 写景状物文：从观察景物的地点变换、时间变化入手分析

议论文

1. 注意文章是以立论为主，还是驳论为主。

2. 注意提出问题、分析问题、解决问题的层次和逻辑关系。

说明文

1. 说明事理和事物结构的，一般按照人们的认识规律和观察顺序安排结构。

2. 介绍前沿科技类的，大多按照“发明—实验—结果”的顺序安排结构。

3. 揭示事物发展过程的，一般按照事物发展进程安排结构。

把握思路

1. 分清层次

包括分清全文的层次和某一部分的层次，注意层次有主次之分，应提取主要的。

2. 摸清各部分之间的关系

主要是开头和结尾与主体部分的关系。

3. 抓题目

题目或表面写作对象，或揭示文章线索，或隐含写作顺序。

4. 抓中心句

中心句往往出现在段首或段尾，抓住中心句就抓住了段落核心。

归纳要点

1. 找出相关的概括性语句。

2. 提取精要，独立归纳。

3. 尽可能用原文词语作归纳。

4. 对相关文字作层次分析，防止要点遗漏。

分析、概括观点与态度

1. 作者的观点态度

（1）观点是作者对事物所持的看法。

（2）态度是作者在文中所表现出的思想倾向和感情倾向，包括肯定与否定、爱与憎、褒与贬。

2. 途径

1. 从概括性强的句子入手

有的文章的观点是直接表述的，抓住了概括性强又表达某种看法的句子，就抓住了作者的观点态度。

2. 从文中运用的材料入手

文中运用的材料，不管是事实还是文献材料总会表达一定的观点，因此从从材料入手是分析概括作者的观点和态度的重要途径。

3. 从作者的评述入手

作者有时候会把自己的观点态度隐藏在具体的评述之中而又不直接说出来，这就要求从分析具体的评述入手，提取精要，作出概括。

4. 抓关键词

抓关键词可以很好地理解作者的态度与看法。
