第三节 教师观
考点
1：教师专业发展

教师劳动特征

1. 复杂性

（1）教育目的的全面性

（2）教育任务的多样性

（3）劳动对象的差异性

2. 创造性

表现在教师创造性的运用教育教学规律：

（1）因材施教

（2）教学方法上的不断更新

（3）教师需要“教育机智”

3. 主体性和示范性

主体性是指教师自身可以成为活生生的教育因素和具有影响力的榜样。

示范性是指教师的言行举止，包括人品、才能、治学态度等都会成为学生学习的对象。

4. 劳动时间和空间的延展性

劳动时间的延续性是由于教师没有固定的工作时间长度，也没有严格的交接班时间界限。空间的广延性是指教师没有严格界定的劳动场所，这是由影响学生发展因素的多样性决定的。

5. 长期性和间接性

长期性是指人才培养的周期比较长，教育影响具有滞后性。间接性是指教师的劳动不直接创造物质财富，而是以学生为中介实现教师劳动的价值。

6. 个体性与群体性

从劳动手段角度来看，教师的劳动主要是以个体劳动的形式进行。同时教师的劳动成果又是集体劳动和多方影响的结果。

教师专业发展的要求

1. 学会学习，成为终身学习者。

2. 勤于反思，成为反思的实践者。

3. 恒于研究，成为教育教学的研究者。

4. 重视沟通，加强交往与合作能力。

5. 勇于创新，培养自己学生的创新能力。

教师专业发展的内容

1. 专业理想的建立。

2. 专业人格的形成。

3. 专业知识的拓展与深化。

4. 专业能力的提高。

5. 专业态度和动机的完善。

6. 专业自我的形成。

教师专业知识的发展

1. 本体性知识

是指教师所具有的任教学科的知识，也称学科专业知识。包括该学科的基本知识、基本技能、学科知识结构体系、学科发展历史及趋势、学科的思维方式和方法论等。

2. 条件性知识

指教师必须具备的教育学、心理学和教育管理知识。

3. 实践性知识

指教师在实现教学目的的教学行为中所具有的课堂情景知识以及相关学科的教学教法知识。

4. 文化知识

是教师专业知识结构的基础，指教师具备的一般人文知识、社会科学知识、自然科学知识，以及基本的艺术素养。

教师专业发展的途径

1. 职前培训

教师从事专业工作的重要基础阶段，是入职前的准备阶段。

2. 入职培训

指经历过教师职前培训的人，在刚获得的教职岗位上，进一步了解岗位知识与能力的培训。

3. 在职培训

教师在自身岗位上不断更新自己的知识、提高专业技能的过程。

4. 自我教育

在教育教学活动中，教师不断进行教学反思，并在实践中提升专业发展技能的过程。自我教育主要有以下几种途径：

（1）专业阅读：教师学习的最主要的形式。

（2）网络学习：随着网络的出现而采用的一种新的学习形式。

（3）校本教研与培训：校本教研与培训是近年来在我国学校出现的一种重要的教师专业发展模式。

（4）行动研究：行动研究是十分有效的和综合的发展途径。
