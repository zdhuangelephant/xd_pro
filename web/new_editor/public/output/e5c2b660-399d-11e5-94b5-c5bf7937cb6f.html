第二节 小学生的学习与指导
考点
2：小学生的学习指导

认知过程的指导

观察的指导

1. 根据要求进行观察。

2. 观察活动与动手操作活动相结合。

3. 知觉过程与思维过程相结合。

4. 掌握相应的观察方法和技术。

注意的指导

1. 明确学习目的，增强注意品质。

2. 采用多种方法和手段集中学生的注意力。

3. 在学生注意力最集中的时间段学习最主要的内容。

思维的指导

1. 善于向学生提问，启发学生提问题，鼓励质疑。

2. 指导学生运用多种思维方法。

3. 发展学生的创新意识，指导学生开展创造性活动。

4. 训练口头语言和书面语言。

记忆的指导

1. 明确记忆任务和目的，积极进行记忆。

2. 正确使用形象记忆等记忆方法。

3. 掌握记忆规律，科学复习。

非认知过程的指导

情绪情感的指导

1. 提高教学质量，丰富活动方式。

2. 创设问题情境，以提问来使学生产生想学习的意愿。

3. 使学生理解学习。

4. 满足学生的求知欲。

意志的指导

1. 通过学习，增强学生学习的主动性、自觉性以及克服困难的勇气。

2. 教给学习方法，培养学习的独立性。

3. 加强指导，培养坚持性。
