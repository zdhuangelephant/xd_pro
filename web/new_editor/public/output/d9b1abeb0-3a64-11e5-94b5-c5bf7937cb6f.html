第一节 小学语文课程标准
考点
5：语文教学与评价建议

教学建议

1. 发挥师生在教学中主动性和创造性。

2. 教学中体现语文的实践性和综合性。

3. 重视情感、态度、价值观的正确导向。

4. 重视培养学生的创造精神和实践能力。

评价建议

1. 充分发挥语文课程评价的多种功能

发挥评价检查、诊断、反馈、甄别、激励、选拔等功能。

2. 恰当运用多种评价方式

形成性评价和终结性评价都是必要的，坚持定性评价与定量评价相结合。

3. 注重评价主体的多元与互动

注意将教师的评价、学生自我评价及学生之间评价相结合。

4. 突出语文课程评价的整体性与综合性

注意识字与写字、阅读、写作、口语交际和综合性学习五个方面有机联系。
