第一节 教师职业道德
考点
2：《中小学班主任工作条例》

《条例》特点

该条例明确了以下关于班主任工作的几个问题：

1. 工作量

明确了班主任的工作量，按当地教师标准课时工作量的一半计入教师基本工作量。

2. 经济待遇

提高了班主任的经济待遇，班主任津贴纳入绩效工资管理。

3. 教育权利

保证了班主任教育学生的权利，可以适当采取批评等方式教育和管理学生。

4. 班主任地位

强调了班主任在学校中的重要地位，有利于稳定班主任队伍、促进班主任专业成长。

《条例》内容

1. 配备与选聘

中小学每个班级应当配备一名班主任，从班级任课教师中选聘，担任时间连续一学年以上。

2. 职责与任务

班主任要做到全面了解班级内每一个学生，做好班级的日常管理工作，开展班级活动，组织做好学生的综合素质评价工作，与任课教师、学生家长和社区等沟通，形成教育合力。

3. 待遇与权力

学校在教育管理工作中应充分发挥班主任的骨干作用，按当地教师标准课时工作量的一半计入教师基本工作量，班主任津贴纳入绩效工资管理，同时，班主任可以适当采取批评等方式教育和管理学生。

4. 培养与培训

教育行政部门和学校应当制定班主任培养培训规划，开展班主任培训工作。教师教育机构应承担班主任培训任务。

5. 考核与奖惩

对有贡献的教师定期给予奖赏，对不能履行班主任职责的，应当调离。

贯彻《条例》要求

• 坚持育人为本，德育为先。

• 注重公平，面向班集体每一位学生。

• 关心学生的全面发展。

• 建立平等互信的师生关系。

• 建立完善的班级管理制度。

• 鼓励学生参加班级和社会实践活动。

《条例》的实施

1. 组织班主任培训

要将中小学班主任培训纳入教师教育计划，有组织地开展岗前和岗位培训，定期交流班主任工作经验，组织班主任进行社会考察，提高班主任的政治素质、业务素质、心理素质和工作及研究能力。

2. 合理安排班主任工作量

保障班主任教师有时间和精力来开展班主任工作。

3. 完善班主任的奖励制度

将优秀班主任的表彰奖励纳入教师、教育工作者的表彰奖励体系之中，定期表彰优秀班主任。

4. 把班主任工作作为学校教育的重要工作

制定切实可行的办法加强班主任工作，建立科学的班主任评价体系，建立班主任工作档案。
